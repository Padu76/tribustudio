# Tribù Radio

Streaming musicale per locali commerciali. Musica originale AI-generated, niente pubblicità, player semplice.

## Setup

```bash
npm install
# configura .env.local con le variabili Supabase e PayPal
npm run dev
```

## Stack

- Next.js 15
- Supabase (database + storage)
- Tailwind CSS
- PayPal (abbonamenti)
